# FAQ

## What does this tool do?

This tool analyzes AI subscriptions and identifies possible savings opportunities based on pricing plans, team size, and use cases.

---

## Do I need an account?

No. Users can run the audit without signup. Email is only needed if they want the report sent or saved.

---

## Which AI tools are supported?

Currently supported tools include:

* ChatGPT
* Claude
* Cursor
* GitHub Copilot
* Gemini
* Windsurf

---

## Are recommendations AI-generated?

The actual savings calculations use rule-based logic for consistency. AI is mainly used for personalized summaries.

---

## Is personal information shown publicly?

No. Public shareable audit links remove sensitive details like email addresses and company names.
